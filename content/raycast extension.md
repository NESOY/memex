# Raycast extension
## How to contribute
1. https://github.com/raycast/extensions 에 접근하여 원하는 extension을 찾는다.
2. 원하는 extension에서 `npm install && npm run dev`를 진행한다.
3. 기능을 변경하고 PR을 요청한다.
