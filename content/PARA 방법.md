# PARA 방법
- 네 가지로 정보를 분류하여 정보를 정리하는 방법
	- Projects : 진행중인 일, Area에서 성취하고자 하는 일
	- Areas : 삶에서 관리하고자 하는 부분(재정, 건강, 학습, 독서 등)
	- Resources : Projects, Area를 성취하기 위한 자료들
	- Archives : 위 세 개의 페이지 중에서 완료하거나, 중단하여 보관하는 페이지
