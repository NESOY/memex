# Raycast auto quit
- 어플리케이션을 특정 시간동안 사용하지 않으면 자동으로 종료되는 기능
- 어지러운 맥을 깔끔하게 유지해주는 기능

## How to use
- 어플리케이션 -> Action -> Enable Auto Quit를 활성화
